## 📖 Descripción

Aplicación web interactiva que simula el crecimiento de plantas dentro de un invernadero.

El usuario puede seleccionar el número de plantas y ver cómo crecen de forma animada hasta alcanzar el techo del invernadero.

---

## 🚀 Características

- 🌿 Selección de plantas (1 a 6)  
- 🎲 Crecimiento aleatorio  
- 🏁 Planta ganadora  
- 📊 Tabla de resultados  
- 🔄 Reinicio de partida
