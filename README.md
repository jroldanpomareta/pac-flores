## 📖 Descripción

Aplicación web interactiva que simula el crecimiento de plantas dentro de un invernadero.

El usuario puede seleccionar el número de plantas (de 1 a 6) y observar cómo crecen de forma animada hasta alcanzar el techo del invernadero.

---

## 🚀 Características

- 🌿 Selección de plantas (1 a 6)  
- 🎲 Crecimiento aleatorio  
- 🏁 Detección de la planta ganadora  
- 📊 Tabla con orden de llegada  
- 🔄 Botón de reinicio (Replantar)  
- 🌱 Animaciones dinámicas con JavaScript

## ⚙️ Funcionamiento

- El usuario selecciona el número de plantas  
- Se generan dinámicamente en el invernadero  
- Cada planta crece con una velocidad aleatoria  
- La primera planta en alcanzar el techo es la ganadora  
- Se muestra una tabla con el orden de llegada  

---

## ▶️ Cómo usar

1. Descargar el repositorio  
2. Abrir el archivo:


index.html


3. Seleccionar número de plantas  
4. Pulsar **Iniciar**
